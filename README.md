# Cloud Quick Start Infrastructure

A cloud infrastructure quick start repository implementing Microsoft's Enterprise Scale Landing Zone patterns with Avanade's proven accelerators and best practices.

## Architecture Principles

This repository adheres to the **Cloud Quick Start Infrastructure Constitution** (`.specify/memory/constitution.md`) which mandates:

- **Enterprise Scale Foundation**: All infrastructure follows Microsoft Enterprise Scale Landing Zone patterns
- **Well-Architected Framework Compliance**: Security, Cost, Operations, Performance, and Reliability pillars
- **Security & Governance First**: Zero Trust, Azure Policy, and compliance controls from day one
- **Avanade Value Acceleration**: Leveraging unique Avanade capabilities and proven accelerators
- **Infrastructure as Code Excellence**: Bicep-first with automated testing and environment parity

## Quick Start

1. Review the constitution: `.specify/memory/constitution.md`
2. Follow Enterprise Scale design patterns for your specific use case
3. Leverage Bicep modules in `infrastructure/bicep/modules/`
4. Ensure compliance with Azure Policy framework
5. Implement monitoring and cost optimization from the start
